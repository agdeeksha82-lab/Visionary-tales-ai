# Visionary-tales-ai
Visionary tales is an Ai storytelling app that generates and narrates stories from uploaded images, supporting multiple genres, languages and age groups.
The Storytelling App transforms images into narrated stories tailored to different genres and age groups. By simply uploading an image, users can experience a unique storytelling journey — whether it’s a whimsical fairy tale for children, a thrilling mystery for teens, or an inspiring narrative for adults.

✨ Features
- Image-based Narration: Upload any image, and the app generates a story inspired by it.
- Genre Selection: Choose from genres like:
- Fantasy
- Adventure
- Mystery
- Sci-Fi
- Age Group Customization:
- Children (simple, fun, imaginative)
- Teens (engaging, adventurous, relatable)
- Adults (thoughtful, deep, genre-rich)
- Voice Narration: Stories are narrated aloud with expressive voices.
- Multi-language Support: Narration available in multiple languages.
- Interactive Mode: Option to let users make choices that influence the story direction.
  
Usage
- Upload an image.
- Select genre and age group.
- Click Generate Story.
- Listen to the narration or read along with the text.
